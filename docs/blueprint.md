# **App Name**: CampusConnect

## Core Features:

- Login & Authentication: Secure user authentication via Firebase, supporting both traditional email/password and Web3 wallet connections using Ethers.js or Web3.js.
- Dynamic Event Feed: Real-time, scrollable feed of events with filtering, search, and detailed event cards including images, titles, dates, and prices.
- NFT Ticket Gallery: Visually appealing gallery displaying user's tickets as unique NFT cards with scannable QR codes, accessible within the wallet.
- Campus Credits Display: Prominently display the user's Campus Credits balance and allow conversion of Campus Credits into cash
- Event Creation Wizard: Multi-step form to guide users through creating new events, setting ticket types and quantities, and publishing events to Firebase via Cloud Functions.
- Ticket Resale Analyzer: Use generative AI as a tool to help estimate and display resale ticket pricing
- Settings Panel: Comprehensive settings panel allowing users to manage their profiles, payment methods, security (password change, 2FA), and app preferences (notifications, dark mode).

## Style Guidelines:

- Primary color: Deep saffron (#FF8F00) to reflect the youthful energy and vibrant community spirit of campus life.
- Background color: Light sand (#F4EDE4) for a warm and inviting feel. The color is close in hue to the primary, but desaturated to 20% saturation and with a brightness of 96%.
- Accent color: Sky blue (#3F51B5) as a contrasting but harmonious tone that provides emphasis without overwhelming. Approximately 30 degrees from primary hue; lower brightness and higher saturation for emphasis.
- Headline font: 'Belleza', a sans-serif with personality and a style aligned to fashion, art, and design
- Body font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel
- Use consistent and clear icons from a standard library (e.g., Material Design Icons) for navigation, actions, and status indicators.
- Mobile-first approach with a bottom-fixed navigation bar for easy access to core features on all devices.
